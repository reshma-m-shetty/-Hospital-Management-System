# 🏥 Healthcare Data Analysis using SQL

## 📌 Overview

This project focuses on analyzing a healthcare dataset using SQL. It explores patient data, doctor performance, appointments, treatments, and billing insights.

The goal is to derive meaningful insights that can help in improving hospital operations and decision-making.

---

## 🚀 Key Features

* Patient demographics analysis
* Doctor performance tracking
* Appointment trends and patterns
* Revenue and billing insights
* Advanced SQL queries using:

  * Joins
  * Window Functions
  * Subqueries
  * Aggregations

---

## 🛠️ Tech Stack

* SQL (MySQL)
* Database Concepts

---

## 📊 Project Highlights

### 👩‍⚕️ Patient Insights

* Total patients count
* Gender distribution
* Patients with frequent visits
* Patients with no appointments

### 🧑‍⚕️ Doctor Analysis

* Doctors per specialization
* Top-performing doctors
* Ranking doctors based on appointments
* Doctors with highest revenue

### 📅 Appointment Analysis

* Monthly trends
* Busiest day of the week
* Peak hour of appointments
* Consecutive patient visits

### 💰 Revenue Insights

* Total revenue generated
* Revenue per patient
* Top paying patients
* Paid vs unpaid percentage

---

## 📈 Advanced Concepts Used

* Views
* Indexing
